# MovieHub

MovieHub is a simple web application that allows users to browse a list of movies and view detailed information about each title.

## Project Structure
```
src/
  ├── assets/
  ├── components/
  ├── contexts/
  ├── layouts/
  ├── pages/
  ├── App.jsx
  └── main.jsx
```
## Run Locally

Clone the project

```bash
  git clone https://github.com/arielmusa/moviehub-frontend.git
```

Go to the project directory

```bash
  cd moviehub-frontend
```

Run the project

```bash
npm install
npm run dev
```

The application will be available at:
http://localhost:5173
